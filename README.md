# Salon Management System (Java Web)

## Overview
A web application for managing salon services, bookings, and administration.

## Tech Stack
- Java
- JSP / Servlets
- HTML, CSS, Bootstrap
- Oracle 11g

## Features
- User registration & login
- Salon browsing
- Service booking
- Admin dashboard
- Manage salons, categories, experts
- Booking management

## Modules
- User Module
- Admin Module
- Salon Owner Module

## Setup Instructions
1. Import project into Eclipse
2. Configure Oracle database
3. Deploy on Apache Tomcat
4. Run the application
